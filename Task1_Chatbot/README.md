## 📌 Description
This project is a simple rule-based chatbot developed using Python and Streamlit.  
It responds to user inputs based on predefined conditions and provides an interactive chat experience through a web interface.

I built this project to understand how basic chatbots work using conditional logic and how user interaction can be handled in a simple application.

---

## 🚀 Features
- Interactive chatbot interface using Streamlit  
- Responses generated using if-else conditions  
- Remembers user name during the session  
- Shows current date and time  
- Handles basic conversational queries  

## 🛠️ Technologies Used
- Python  
- Streamlit
