## 📌 Description
This project is a simple Tic-Tac-Toe game developed using Python, where a human player competes against an AI.

The AI is implemented using the Minimax algorithm, which allows it to evaluate all possible moves and choose the best one. This ensures that the AI plays optimally and is difficult to beat.

I built this project to understand how decision-making works in games and how algorithms can be used to simulate intelligent behavior.

## 🚀 Features
- Command-line based gameplay  
- AI opponent using Minimax algorithm  
- Detects win, loss, and draw conditions  
- Ensures optimal moves by the AI  
- Simple and clear game flow  

## 🛠️ Technologies Used
- Python
