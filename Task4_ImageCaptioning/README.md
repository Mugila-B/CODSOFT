📌 Description

This project is an image captioning system built using Python and deep learning models. It generates meaningful textual descriptions for images.

The system uses a pre-trained BLIP model to extract visual features from images and convert them into natural language captions.

I built this project to understand how computer vision and natural language processing can be combined to describe visual content automatically.

🚀 Features

Generates captions for input images
Uses pre-trained BLIP model
Combines computer vision and NLP
Produces human-readable descriptions
Simple and effective implementation
