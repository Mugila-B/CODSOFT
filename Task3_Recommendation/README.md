## 📌 Description
This project is a simple recommendation system built using Python. It suggests similar movies based on their genres.

The system uses basic text processing and similarity techniques to find related movies and provide recommendations.

I built this project to understand how recommendation systems work and how similarity between items can be calculated.

## 🚀 Features
- Recommends movies based on similarity  
- Uses content-based filtering  
- Simple and easy-to-use interface  
- Demonstrates basic recommendation logic  

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn
