📌 Description

This project is a face detection and recognition system built using Python and OpenCV. It detects human faces in images and identifies the person using a trained model.

The system uses Haar Cascade for face detection and the LBPH algorithm for recognizing faces based on trained data.

I built this project to understand how computer vision systems detect and recognize human faces using pre-trained models and image processing techniques.

🚀 Features

Detects faces in images
Recognizes faces using trained dataset
Uses Haar Cascade for detection
Uses LBPH algorithm for recognition
Displays name with bounding box

🛠️ Technologies Used

Python
OpenCV (opencv-contrib-python)
NumPy
